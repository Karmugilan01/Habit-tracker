This repository is a starter template for a Habit Tracker application built with React and Vite.
Currently it contains only the boilerplate setup. The project is ready for you to implement habit tracking features such as:

*Displaying a list of habits

*User interactions with checkboxes

*Progress tracking UI

*Reset functionality
